Reconnect - Em produção

Javascript / React / NodeJS / Express / MongoDB / Web scraping...

Reconnect

  O projeto surgiu da necessidade de aprimorar o gerenciamento de devoluções de mercadorias na empresa, evoluindo para um aplicativo de gerenciamento geral de estoque, composto por quatro páginas principais:

Chat: Integrado com a API do Mercado Livre, semelhante ao WhatsApp, com temporizador de mensagens para melhorar a comunicação com os clientes e monitorar o tempo de resposta.

Estoque Mínimo: Exibe produtos com estoque baixo, facilitando o controle de itens em falta e permitindo uma gestão mais eficiente.

Pedidos: Coleta dados diários e gera gráficos, oferecendo uma visão clara da demanda de produtos e ajudando na análise de tendências.

Devolução: Substituiu o antigo sistema baseado em Excel, proporcionando um controle automatizado e visualmente eficiente de cada caso.
